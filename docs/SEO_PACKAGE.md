## 5. FAQ (SEO BOOST)

Use `FAQPage` schema where possible.

**Q1: What is MMD online?**  
MMD online means running MikuMikuDance-style workflows in a web browser — loading PMX models and VMD motions without installing desktop MMD. AnimaStage Lite is a full browser studio with physics, timeline, and video export.

**Q2: Can I run MikuMikuDance in the browser?**  
You cannot run the official MMD executable in a browser, but AnimaStage Lite supports the same file formats (PMX/PMD, VMD) and lets you preview, edit, and export video online using WebGL2.

**Q3: Does AnimaStage Lite work without install?**  
Yes. Open animastage-lite.app/app in Chrome or Edge — no download required. An optional Android APK is available if you prefer a native app icon on your phone.

**Q4: Is AnimaStage Lite free?**  
The Lite studio is free to use in the browser. Open source on GitHub. Optional AI features may require your own API key. AnimaStage Pro is a separate paid/advanced product.

**Q5: What file formats are supported?**  
PMX and PMD models, VMD motion (including camera tracks), textures, and HDR environments. Export includes MP4 (WebCodecs on Chrome/Edge) and VMD from the timeline editor.

**Q6: Can I export vertical video for TikTok and YouTube Shorts?**  
Yes. Switch to 9:16 portrait mode and export at 1080×1920. The studio optimizes performance for stable vertical recording.

**Q7: Do my models get uploaded to a server?**  
Core editing runs client-side in your browser. Files are not uploaded for basic load-and-play unless you enable optional cloud/collab features.

**Q8: MMD online vs desktop MMD — which should I use?**  
Use AnimaStage Lite for quick previews, Shorts, and any device with a modern browser. Use desktop MikuMikuDance for legacy plugins, maximum compatibility, and long-form traditional MMD production.

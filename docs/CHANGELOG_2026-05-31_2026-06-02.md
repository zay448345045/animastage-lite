## 31 May 2026 — Import & HUD (from mmd_rtx)

### Multi-character import (2+)
- Drop / folder / **ZIP** → up to **4** `.pmx` / `.pmd` models in one upload.
- Auto placement on scene slots (`getNextSpawnPosition`, ±7 m for duets).
- **VMD** files distributed round-robin (1 VMD → 1st character, 2 VMDs → one per character, etc.).
- Second character: `patchStateForMultiCharacterLoad` (physics `playtime` mode).

### Import formats (`ingestAssetBundle`)
| File | Role |
|------|------|
| `src/utils/assetImport.ts` | ZIP, PMX/PMD/VMD/HDR/textures, model dedup |
| `src/utils/mmdFiles.ts` | `buildFileMapFromFiles`, export `classifyVmdFiles` |
| `src/components/FileUploader.tsx` | Single upload entry point |
| `src/App.tsx` | `handleLoadCustomModel` — array of `ProcessedMMDFiles` |

Supported in a bundle: `.pmx`, `.pmd`, `.vmd`, `.zip`, textures, `.hdr`; only **PMX/PMD** spawn as dancers in the scene.

### Performance HUD
| File | Shows |
|------|--------|
| `src/utils/sceneStats.ts` | Scene triangle count |
| `src/components/ViewportPerfMonitor.tsx` | Smoothed FPS + tris |
| `src/components/Viewport.tsx` | FPS / tris / autoScale overlay |
| `src/product/ui/PerformanceOverlay.tsx` | Product-style overlay |

### Camera (duo groundwork)
- `src/scene/characterHeadRegistry.ts` — root registration, head, duo target.
- `src/scene/cameraFraming.ts` — `single` / `duo`, `getStageTargetVector`.
- `src/context/CameraFollowContext.tsx` — follow API for MMD mode.

## 2 June 2026 — Shorts, camera, export

### Generate Short (VMD preserved)
| Change | File |
|--------|------|
| Dedicated pipeline, VMD kept | `product/shorts/applyShortsPipeline.ts` |
| Does not call full `templateEngine.apply` for motion | `ShortsGenerator.ts` |
| Dialog: duration 20–90 s, presets | `product/ux/ShortsSetupDialog.tsx`, `shortsConfig.ts` |
| VMD per character + Add VMD | `useProductLayer.appendModelVmdFiles` |
| Preview: Auto frame / Manual / Export | `ShortsFlowBar.tsx`, `ProductShortsFlow.tsx` |

### Camera — follow & manual mode
| Issue | Fix |
|-------|-----|
| Character left 9:16 frame / export | `StageAutoFollow` — snap while recording |
| Orbit locked around (0,10,0) | `cameraOrbitAnchor` + `offsetCameraSnapshotToFocus` |
| MMD + Bloom framed on hand | lookAt on character focus, not VMD `cameraTarget` |
| User-placed camera in MMD | `manualCameraLock` + Orbit in MMD, **Manual** button |

Key files:
- `product/camera/StageAutoFollow.tsx`
- `product/camera/frameShortCamera.ts`
- `scene/cameraFocus.ts`
- `components/MMDCameraController.tsx`
- `components/PortraitCameraFraming.tsx` (does not block auto-focus)

### MP4 export (WebCodecs)
- `VideoFrame.close()` + drain `encodeQueueSize`.
- Recreate `VideoEncoder` on `QuotaExceededError` / codec reclaimed.
- `useVideoRecorder` — 3× `requestAnimationFrame` before each captured frame.
